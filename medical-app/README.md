# AI Medical Report Analyzer

## Sprint 1 - Experiment 7

### User Stories Implemented
- **US-01**: User Registration & Login (Firebase Auth)
- **US-02**: Upload Medical Report (PDF, JPG, PNG)
- **US-03**: OCR Text Extraction

### How to Run

1. Install dependencies:
```
npm install
```

2. Start the app:
```
npm start
```

3. Open browser at:
```
http://localhost:3000
```

### Tech Stack
- React.js
- Firebase Authentication
- Firebase Firestore Database
